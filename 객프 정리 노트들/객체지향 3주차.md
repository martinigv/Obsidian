## 16Slide
1. class 이름은 Rectangle
2. width, height int타입의 맴버 변수 필요
3. getArea() 맴버 함수 필요
```c++
#include <iostream>
using namespace std;
class Rectangle {
public:
	int width;
	int height;

	int getArea() {
		return width * height;
	}
};


int main() {
	Rectangle rect;
	rect.width = 3;
	rect.height = 5;
	cout << "사각형의 면적은 " << rect.getArea() << endl;
}
```
